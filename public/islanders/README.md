# Islander Images

Drop local islander preview photos in this folder.

Use lowercase first-name filenames:

- `bryce.jpg`
- `trinity.jpg`
- `aniya.jpg`
- `kenzie.jpg`
- `amaya.jpg`
- `bryan.jpg`

The app looks for `/public/islanders/{display-name}.jpg`. If an image is missing, it shows a clean initial avatar instead.

For names with spaces, use hyphens, for example `maya-s.jpg`.
